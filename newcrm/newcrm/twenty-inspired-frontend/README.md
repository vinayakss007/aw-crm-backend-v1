# Abetworks CRM - Twenty Inspired Frontend

This is a modern CRM frontend inspired by the Twenty CRM design and architecture. It features a clean, responsive interface with dashboard, contacts, companies, and opportunities management.

## Features

- Responsive layout with sidebar navigation
- Dashboard with analytics and charts
- Contacts management
- Companies management
- Opportunities tracking
- Modern UI with Tailwind CSS
- React with TypeScript
- Vite for fast development

## Tech Stack

- React 18
- TypeScript
- Vite
- Tailwind CSS
- React Router
- Recoil for state management
- Recharts for data visualization
- Heroicons for icons

## Getting Started

1. Install dependencies:
```bash
npm install
```

2. Start the development server:
```bash
npm run dev
```

3. The application will be available at `http://localhost:3001` (or another available port)
